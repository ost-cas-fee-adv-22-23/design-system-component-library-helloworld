# Typography
As primary font the google font "Poppins" is used. For Mumble the default font is overwritten with "Poppins" and it does not need to used it explicitly. It is one of the fonts that contains a wide range of readability, so it is very common to see it in continuous text and in large headings.

# Type scale
Type scales are hierarchical type systems consisting of style definitions for common elements.

| Scale                             | Weight                 | Font-Family             | Size                  |
| --------------------------------- | ---------------------- | ----------------------- | --------------------- |
| head-1                            | bold       (700)       | Poppins                 | l3    (3rem/48px)     |
| head-2                            | bold       (700)       | Poppins                 | l2    (2.5rem/40px)   |
| head-3                            | semibold   (600)       | Poppins                 | l1    (2rem/32px)     |
| head-4                            | semibold   (600)       | Poppins                 | m2    (2rem/24px)     |
| paragraph-L                       | medium   (500)         | Poppins                 | m2    (2rem/24px)     |
| paragraph-M                       | medium   (500)         | Poppins                 | s3    (1.125rem/18px) |
| label-XL                          | semibold   (600)       | Poppins                 | m2    (2rem/24px)     |
| label-L                           | semibold   (600)       | Poppins                 | m1    (1.25rem/20px)  |
| label-M                           | semibold   (600)       | Poppins                 | s2    (1rem/16px)     |
| label-S                           | semibold   (600)       | Poppins                 | s1    (0.875rem/14px) |
