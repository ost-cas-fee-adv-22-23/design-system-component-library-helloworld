# Typography
As primary font the google font "Poppins" is used. For Mumble the default font is overwritten with "Poppins". It is one of the fonts that contains a wide range of readability, so it is very common to see it in continuous text and in large headings.


# Type scale
Type scales are hierarchical type systems consisting of style definitions for common elements. The existing components headers 'h1', 'h2' etc., 'p' and 'label' are set with styles which used by Mumble.
The  scales can be overridden if it is necessary.

| Scale                             | Weight                 | Font-Family             | Size                  |
| --------------------------------- | ---------------------- | ----------------------- | --------------------- |
| h1                                | bold       (700)       | Poppins                 | l3    (3rem/48px)     |
| h2                                | bold       (700)       | Poppins                 | l2    (2.5rem/40px)   |
| h3                                | semibold   (600)       | Poppins                 | l1    (2rem/32px)     |
| h4                                | semibold   (600)       | Poppins                 | m2    (2rem/24px)     |
| p (l)                             | medium   (500)         | Poppins                 | m2    (2rem/24px)     |
| p (m)                             | medium   (500)         | Poppins                 | s3    (1.125rem/18px) |
| label (xl)                        | semibold   (600)       | Poppins                 | m2    (2rem/24px)     |
| label (l)                         | semibold   (600)       | Poppins                 | m1    (1.25rem/20px)  |
| label (m)                         | semibold   (600)       | Poppins                 | s2    (1rem/16px)     |
| label (s)                         | semibold   (600)       | Poppins                 | s1    (0.875rem/14px) |

## Usage
You can use the known components. The scale of the component is set by default for mumble.
```jsx
<h1>Heading 1</h1>
<h2>Heading 2</h2>
```
If you want to use the component with another height or weight, just define it on the component.
```jsx
<h1 className="text-l1 font-semibold">Heading</h1>
```

# Paragraph
**Scale:** p (l), p (m)
**Default:** p(m) is the default p

<Typeset
    fontSizes={[
        Number(typography.fontSize.m2),
        Number(typography.fontSize.s3)
    ]}
    fontWeight={fullConfig.theme.fontWeight.medium}
    sampleText={SampleText}
    fontFamily={fullConfig.theme.fontFamily.sans}/>

## Usage
Here the HTML element p has the expected css styling. The default styling ist the size 'm'. If you need another size just added to the element.
```jsx
<p>Paragraph M</p>
// With larger size
<p className="text-m2">Paragraph L</p>
```

# Label
**Scale:** label (xl), label (l), label (m), label (s)
