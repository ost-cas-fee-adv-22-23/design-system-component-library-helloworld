## Color Slate
The color Slate us used as primary text color.

## Color Violet
The color Violet is used mostly for hover, interaction, navigation and clickable or clicked states of an element.

## Color Pink
The color Pink is used as secondary interaction for elements.

## Color Black
The color Black is selectable but is not defined for Mumble interactions.

## Color White
The color White is used as background color and for text if dark mode is used.

## Color Red
The color Red is used for warnings and error messages.
