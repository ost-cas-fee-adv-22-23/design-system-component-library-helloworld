# Iconography
## How to use icon components
Icons inherit the text-color of the parent as shown in the following example.
Some of the icons do additionaly to the default variant have a filled variant, too.

### Available Icons with no "fill" variant

#### Example code:

     <div className={'text-pink-500 hover:text-violet-900'}>

</div>
